🔧 簡單登入測試

目標：繞過重導向問題，直接測試後端登入功能

方法 1：使用 Google OAuth Playground

我們可以使用 Google 的官方工具來獲取 ID Token，然後直接測試後端：

  1. 前往 Google OAuth 2.0 Playground
  2. 在左側選擇 "Google OAuth2 API v2"
  3. 勾選 "https://www.googleapis.com/auth/userinfo.email" 和 "https://www.googleapis.com/auth/userinfo.profile"
  4. 點擊 "Authorize APIs"
  5. 完成 Google 登入
  6. 點擊 "Exchange authorization code for tokens"
  7. 複製 "id_token" 的值
